This is a pre-compiled program for Windows.
In normal mode, Output files are stored in Images\Output\
In silent mode, Output files are stored in Output\Images\
When running in normal mode, please put images in Images directory but not include folder name in the argument.
Run instruction:

Normal:
OpenCVProject.exe filename.jpg

Silent mode:
OpenCVProject.exe filename.jpg noshow

Check all image files in Images folder:
run.bat
